# Vocab Trainer — 30 Day Challenge

Master 150 words in 30 days with 4 exercise modes per day.

---

## 🚀 Deploy from your iPhone (step by step)

### Step 1: Create a GitHub account
1. Open Safari → go to **github.com**
2. Tap **Sign Up** and create a free account
3. Verify your email

### Step 2: Create a new repository
1. Go to **github.com/new**
2. Name it `vocab-trainer`
3. Keep it **Public**
4. Tap **Create repository**

### Step 3: Upload files
1. On the repo page, tap **"uploading an existing file"** link
2. Upload ALL files from this project folder:
   - `package.json`
   - `vite.config.js`
   - `index.html`
   - `src/main.jsx`
   - `src/App.jsx`
   - `public/manifest.json`
   - `public/icon.svg`
   - `public/favicon.svg`

**⚠️ Important:** GitHub's mobile upload is flat (no folders). 
You'll need to create files manually instead:

1. On your repo page, tap **Add file → Create new file**
2. For files in `src/` folder, type the path as: `src/main.jsx`  
   (typing the `/` will auto-create the folder)
3. Paste the file contents
4. Tap **Commit changes**
5. Repeat for each file

### Step 4: Deploy on Vercel
1. Open Safari → go to **vercel.com**
2. Tap **Sign Up** → choose **Continue with GitHub**
3. Authorize Vercel to access your GitHub
4. Tap **Add New → Project**
5. Find `vocab-trainer` and tap **Import**
6. Leave all settings as default (Vercel auto-detects Vite)
7. Tap **Deploy**
8. Wait ~60 seconds — done! You'll get a URL like `vocab-trainer.vercel.app`

### Step 5: Add to iPhone Home Screen
1. Open your new URL in Safari
2. Tap the **Share** button (box with arrow)
3. Scroll down and tap **Add to Home Screen**
4. Name it "Vocab" or whatever you like
5. Tap **Add**

Now it looks and feels like a native app! 🎉

---

## What's inside

- **150 words** across 30 themed categories
- **4 exercise modes** per day: Learn → Match → Fill In → Quiz
- **Progress tracking** with streaks, mastery %, and time spent
- **Persistent data** — your progress saves to your phone
- Works offline once loaded
